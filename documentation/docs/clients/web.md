# Web

> Query your Second Brain from your Web Browser

Start chatting with Apollos right from your web browser. You can upload documents directly via the web UI, or sync your data using [Obsidian](/clients/obsidian), [Notion](/data-sources/notion_integration), or [Github](/data-sources/github_integration).

Go see it here --> [Apollos Cloud](https://app.apollosai.dev).

![Apollos Web App Home](/img/apollos_web_app_home.png)

## Features
- **Chat**
  - **Faster answers**: Find answers quickly, from your private notes or the public internet
  - **Assisted creativity**: Smoothly weave across retrieving answers and generating content
  - **Iterative discovery**: Iteratively explore and re-discover your notes
- **Search**
  - **Natural**: Advanced natural language understanding using Transformer based ML Models
  - **Incremental**: Incremental search for a fast, search-as-you-type experience

## Setup
No setup required. The Apollos web app is the default Apollos client. You can access it from any web browser. Try it on [Apollos Cloud](https://app.apollosai.dev)

## Upload Documents
You can upload documents to Apollos from the web interface, one at a time. This is useful for uploading documents from your phone or tablet. To upload a document:
1. You can drag and drop the document into the chat window.
2. Or click the paperclip icon in the chat window and select the document from your file system.

![Demo of dragging and dropping a file](https://assets.apollosai.dev/home_page_data_upload.gif)

### Install on Desktop
You can install Apollos as a [Progressive Web App (PWA)](https://web.dev/learn/pwa/installation) on your desktop for quick access.

1. Open [Apollos Cloud](https://app.apollosai.dev) or your self-hosted Apollos server in Chrome
2. Click the install button in the address bar to install the app

![progressive web app install icon](/img/pwa_install_desktop.png)

Alternatively:
1. Open the three-dot menu in the top right corner of Chrome
2. Go to 'Cast, Save, and Share' option
3. Click on the "Open in Apollos" option

### Install on Phone
You can also install Apollos as a PWA on your phone for quick access.
1. Login to [Apollos Cloud](https://app.apollosai.dev) or your self-hosted Apollos server from the web browser (prefer Chrome/Edge) on your phone
2. Open the 3 dot menu on the browser and click the "Add to Home screen" option
3. Click "Install" on the next screen to add the Apollos icon to your phone Home screen
