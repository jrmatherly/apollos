# Web

> Query your Second Brain from your Web Browser

Without any desktop clients, you can start chatting with Apollos on the web. Bear in mind you do need one of the desktop clients in order to share and sync your data with Apollos.

Go see it here --> [Apollos Cloud](https://app.apollos.dev).

![](/img/apolloslos_web_app_home.png)

## Features
- **Chat**
  - **Faster answers**: Find answers quickly, from your private notes or the public internet
  - **Assisted creativity**: Smoothly weave across retrieving answers and generating content
  - **Iterative discovery**: Iteratively explore and re-discover your notes
- **Search**
  - **Natural**: Advanced natural language understanding using Transformer based ML Models
  - **Incremental**: Incremental search for a fast, search-as-you-type experience

## Setup
No setup required. The Apollos web app is the default Apollos client. You can access it from any web browser. Try it on [Apollos Cloud](https://app.apolloslos.dev)

## Upload Documents
You can upload documents to Apollos from the web interface, one at a time. This is useful for uploading documents from your phone or tablet. To upload a document:
1. You can drag and drop the document into the chat window.
2. Or click the paperclip icon in the chat window and select the document from your file system.

![demo of dragging and dropping a file](https://assets.apolloslos.dev/home_page_data_upload.gif)

### Install on Phone
You can optionally install Apollos as a [Progressive Web App (PWA)](https://web.dev/learn/pwa/installation). This makes it quick and easy to access Apollos on your phone.
1. Login to [Apollos Cloud](https://app.apolloslos.dev) or your self-hosted Apollos server from the web browser (prefer Chrome/Edge) on your phone
2. Open the 3 dot menu on the browser and click the "Add to Home screen" option
3. Click "Install" on the next screen to add the Apollos icon to your phone Home screen
